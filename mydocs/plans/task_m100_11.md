## 목표 아키텍처

### 설계 원칙

- NAT Gateway 전면 제거 → 양 환경 모두 NAT Instance (NatInstanceProviderV2, Amazon Linux 2023)
- Aurora Serverless v2 제거 → 양 환경 모두 RDS PostgreSQL db.t3.micro (Single-AZ)
  - dev: 20GB / prod: 50GB (운영 데이터 여유 확보)
- CloudWatch 로그 보존: dev 7일 / prod 14일
- Naver/NCP 자격증명: SSM Parameter Store (기존 유지)
